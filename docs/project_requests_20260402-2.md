以下是正常的玩家游戏流程，这个游戏流程就是本游戏的大部分玩法，如果后期有变动再新增玩法。主要流程如下：
1、点击游戏开始，根据游戏难度，在全局卡池里随机抽取对应难度的4张卡牌作为对手；暂时不设计BOSS战；
2、等待玩家选牌，玩家选牌结束
3、先计算双方选牌自己的主被动技能（如有），再计算选牌人物之间的羁绊（需要另外建羁绊表，最终还是用技能，这里就是隐藏技能，因为人物羁绊这块我想隐藏）
4、游戏为回合制，骰子决定先后手（骰子比大小），骰子点数多少与双方速度总和有关（属性速度的作用）
5、先手的玩进攻类小游戏，后手的玩防守型小游戏，小游戏有很多种类，可以参照瓦利奥制造。小游戏需要单独做模块的，因为以后小游戏会很多，会从里面随机抽取，然后带入难度参数（这个难度参数就取决于自己属性和对手属性了，武力对应小游戏得分，比对手越高得分越高，智慧决定小游戏难易程度，比对手越高越容易）。小游戏一般游玩时间5秒，倒计时到0后结算。全部达成目标算完美，有点小失误算棒，等等，带出档次结果，用于后续技能判定。因为出4张牌，所以每方都是玩4个小游戏。
6、双方小游戏结束，开始回合结算，根据技能判定和先后触发开始结算，所以需要动用技能系统，结算结果分别对对方卡牌造成伤害。防御决定伤害的大小，防御比对手越高，伤害越小。
7、精神则是用于特殊攻击的抵抗判定，是非数值伤害类的抵抗，精神比对手越高，则越能抵抗住特殊攻击。（目前举例眩晕、混乱、失明、中毒等特殊状态）
8、攻守互换，玩4个小游戏，结算
9、回到4第四步，骰子决定先后手
10、注意，4个小游戏分别对应4个卡片，从左往右，所以在结算时也要分开结算
11、当有卡片血量归0时，该卡片不能战斗了，小游戏还是玩4次，判定则是使用此卡片左边张卡片作为判定。
12、当所有卡片血量都归0，则游戏结束，结算。
好的，游戏基本围绕此打法进行的，先搭建框架，并能基本运行测试起来。初期小游戏比较少，进攻类和防守类小游戏都只做一个，进攻类的做一个射箭小游戏，防守类做接球小游戏。
