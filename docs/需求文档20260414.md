# 修改优化
## 卡牌制作优化
1. 在卡牌制作-成品卡牌预览里，应该预览三种，一种是原图750px*1125px的，通过点击原卡预览弹窗预览，第二个是约1/2大小的，直接在成品卡牌这个窗口预览，第三种是缩略版，是原图的约1/4大小，这三种档次就三种档次，没有其他比例。
2. 其他地方需要用这套方法渲染成品卡牌，注意比例，字比例大小，位置比例大小都要匹配，该显示就显示，没显示的不用显示
3. 至少在对战测试中，选卡界面使用1/4大小，对战界面是1/2大小，需要做相应调整

## 对战测试优化
1. 对战测试是窗口的，右上角要有关闭功能
2. 掷色子后不需要再点击继续了，等待2秒后继续吧（等待不需要显示，自动开始）
3. 战斗日志需要全面，能观察到，可以滚轮
4. 对战测试的窗口里，卡牌是原图1/2大小
5. 对战顺序问题：1对1，2对2，3对3，4对4，有卡牌血量清空阵亡后，有以下情况，需要调整对战逻辑：
  - 1对1，1阵亡后，赢的1对2，输的一方还是正常对局，下同
  - 2阵亡，赢的2对3
  - 3阵亡，赢的3对4
  - 4阵亡，赢的4对3
  - 24阵亡，赢的24对对方的13
  - 234阵亡，赢的一边就全对1
  - 输的1234剩余的对剩余的，比如3输了，2还是对2不动
