 # 卡牌制作 -已完成
 kayou管理里左边增加卡牌制作目前思路如下：
  - 选择一个人物，然后展示一下已有的人物内容属性和卡牌图片
  - 这里的思路是一个大的画框，里面展示这个卡牌图片
  - 人物的名字、背景介绍、卡牌编号、系列、阵营、稀有度、4维属性、技能名字与描述，这些都列在那里，然后有多选框
  - 字体可以做选择，要符合游戏的特征，网上找找游戏特色的字体，每个展示的多选框/属性都可以选自己的字体
  - 多选框选上后，画框里就有矩形框，可以调大小，里面的字也跟着调大小，调整位置
  - 成品预览框可以看到效果
  - 有些稀有度卡牌需要上特效，比如镭射、闪金、浮雕等效果，也可以做选择（单选多选）
  - 以上这些选好后需要保存，这样我们这整个平台的卡牌，就做成这个样子
  - 后期在做另外个网页时候，就是需要用到这里的卡牌制作与渲染。所以这里后期会被经常给其他函数程序或者代码调用，挺重要的，需要单独做功能
