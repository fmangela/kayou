│   └── assets/             # 图片存储

## 常用命令

### 开发
```bash
# 启动开发环境（前后端同时启动）
./run-dev.sh

# 或分别启动
cd backend && npm run dev    # 后端 http://localhost:3174
cd frontend && npm run dev   # 前端 http://localhost:5174
```

### 生产部署
```bash
# 一键部署
./deploy.sh

# 或手动部署
npm install                  # 安装依赖
cd frontend && npm run build # 构建前端
cd backend && npm start      # 启动后端
```

### 数据库
```sql
-- 连接数据库
mysql -u kayou -p kayou

-- 查看表结构
SHOW TABLES;
DESC characters;
DESC card_attributes;

-- 重置数据（谨慎操作）
DROP TABLE characters;
DROP TABLE card_attributes;
-- 重启后端会自动重建表结构
```

## API 端点速查

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/auth/login` | POST | 登录 |
| `/api/characters` | GET/POST/PUT/DELETE | 人物管理 |
| `/api/characters/import` | POST | 批量导入 |
| `/api/characters/export` | GET | 批量导出 |
| `/api/llm/generate` | POST | AI 描述生成 |
| `/api/llm/batch-prompt` | POST | 批量提示词生成 |
| `/api/drawing/generate` | POST | AI 绘图 |
| `/api/drawing/status/:id` | GET | 绘图进度 |
| `/api/images/crop` | POST | 图片裁切 |
| `/api/images/convert-webp` | POST | WebP 转换 |
| `/api/cards/attributes` | GET/POST/PUT | 卡牌属性 |
| `/api/cards/design` | POST | 卡牌制作 |
| `/api/games` | GET/POST/PUT | 小游戏配置 |

**认证方式**：所有接口（除登录外）需在 Header 中携带 `Authorization: Bearer <token>`

## 数据库表结构

### characters（人物表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| name | VARCHAR(100) | 姓名 |
| pinyin | VARCHAR(100) | 拼音 |
| background | TEXT | 背景描述 |
| appearance | TEXT | 外形描述 |
| nb2_prompt | TEXT | Nano Banana2 提示词 |
| mj_prompt | TEXT | Midjourney 提示词 |
| image_paths | TEXT | 原图路径 (JSON) |
| webp_paths | TEXT | WebP 路径 (JSON) |

### card_attributes（卡牌属性表）
| 字段 | 类型 | 说明 |
|------|------|------|
| character_id | INT | 外键关联 characters.id |
| card_code | VARCHAR(20) | 卡牌编号 |
| series_name | VARCHAR(50) | 系列名 |
| faction_name | VARCHAR(50) | 阵营 |
| rarity | VARCHAR(20) | 稀有度 |
| force/intellect/speed/stamina_value | INT | 四属性值 |
| element_name | VARCHAR(20) | 元素 |
| skill1/2/3_id | VARCHAR(20) | 技能 ID |
| skill1/2/3_name | VARCHAR(50) | 技能名 |
| skill1/2/3_desc | TEXT | 技能描述 |

### games（小游戏表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| game_key | VARCHAR(64) | 游戏标识（唯一） |
| name | VARCHAR(255) | 游戏名称 |
| formula_config | LONGTEXT | 公式配置（JSON） |

## 常见问题

### Q: 后端启动失败
```bash
# 检查端口占用
lsof -i :3174

# 检查数据库连接
mysql -u kayou -p -e "SELECT 1"

# 查看日志
tail -f logs/combined.log
```

### Q: 前端无法连接后端
- 检查 `frontend/src/api/*.js` 中的 `BASE_URL` 配置
- 开发模式默认 `http://localhost:3174/api`
- 生产模式使用相对路径 `/api`

### Q: 图片上传失败
- 检查 `backend/assets` 目录权限
- 确保磁盘空间充足
- 检查图片格式和大小限制

### Q: AI 绘图无响应
- 检查 MX API 配置（在 settings 表中）
- 查看 `logs/combined.log` 错误信息
- 确认网络可访问外部 API

## 扩展新模块

### 添加新路由
1. 创建 `backend/routes/xxx.js`
```javascript
const express = require('express');
const router = express.Router();
const { pool } = require('../db/init');

router.get('/xxx', authMiddleware, async (req, res) => {
  const [rows] = await pool.query('SELECT * FROM xxx');
  res.json(rows);
});

module.exports = router;
```

2. 在 `server.js` 注册
```javascript
app.use('/api/xxx', authMiddleware, require('./routes/xxx'));
```

### 添加前端页面
1. 创建 `frontend/src/views/admin/XxxView.vue`
2. 在 `router/index.js` 添加路由
3. 在 `AdminLayout.vue` 添加菜单项
