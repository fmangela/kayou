# 对战测试
对战为本卡牌游戏的核心玩法，在kayou管理页面左栏小游戏制作下面，新增对战测试
本游戏主要是跟电脑打对战，PVP玩法后期开发

## 界面介绍
因为是测试，仅测试对战功能是否有效
1. 第一块帮电脑的4张选牌，点击选择卡牌，弹出选择卡牌对话框，按1234顺序选取4张卡牌，默认第一张是队长牌。对话框可以做查询筛选排序功能，翻页显示，跟平常一样
2. 选择完后卡牌的“成品”就从左往右按顺序摆在第二块。这里的成品就是卡牌制作后成品预览的成品，所以需要卡牌制作里的那套已经编好的直接调用。
3. 当然如果顺序不对，还可以拖动卡牌做位置调整
4. 第三块是帮玩家自己选4张卡牌，同1
5. 第四块是列出玩家的4张卡牌，同23
6. 第五块是按钮开始测试，即开始游戏


## 游戏介绍
游戏顺序：
1. 展示游戏双方卡牌
2. 公平起见，掷骰子决定先后手，骰子1-20，要有动画效果
3. 先手玩4局小游戏，随机小游戏，分别对应牌位1234
4. 根据游戏结果按顺序结算
5. 攻守互换，后手玩4局小游戏，随机的，分别对应牌位1234
6. 后手结算
7. 为了方便快速，小游戏可同时进行，当然，大部分时间是和电脑对战的，电脑玩的结果也要按顺序显示
8. 上面是攻守一轮结束，然后继续投骰子
9. 有卡牌先没体力了，就与它左边的卡牌结算，攻击也是它左边的卡牌攻击
10. 一方所有卡牌都没体力了，游戏结束，返回对局结果
