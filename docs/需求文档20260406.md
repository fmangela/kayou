# 小游戏
本 子项目 是kayou卡牌项目里另外一块重要的部分

## 位置
在kayou管理左边卡牌制作下面，添加小游戏制作

## 功能
里面是小游戏的制作完的测试。后期是要用到其他应用的，所以这个小游戏模式要搭框架，每个小游戏都要做一个模块，方便后期调用和修改

## 布局与参数
 - 第一行下拉菜单，选择小游戏，旁边是游玩测试按钮，恒定显示
 - 第二行是卡牌属性参数，我方武力，我方智力，我方速度，我方体力，敌方武力，敌方智力，敌方速度，敌方体力，可以输入数值，恒定显示。并且附上占位符表达以及说明
 - 第三行是影响小游戏的参数，显示与否跟每个小游戏相关
 - 第四行是每个参数的来源公式，这个也是可编辑输入，有占位符，就占位第二行的这些参数，这些参数最后的数值不是手输的，就是通过敌我卡牌属性参数套入公式计算得到的结果，显示与否也跟每个小游戏有关
 - 第五行是保存参数对应的公式配置，以便以后小游戏在真正使用环境下能使用这些公式计算游戏的参数难度，显示与否跟每个游戏相关
 - 第五行就是测试游戏按钮，显示与否跟每个游戏相关。

## 小游戏运行与结算
 - 小游戏的难度参数就是上面布局里的参数，每个小游戏会详细介绍参数与难度
 - 点击测试游戏后，小游戏是弹出对话框形式运行
 - 所有小游戏都一样，进入后，标题与游戏玩法简介，显示3秒，倒计时3秒共6秒
 - 游戏结束后弹出窗口提示游戏得分，得分有5个档次，或者说是5分，这个得分是需要返回的，后期需要这个得分值。

## 目前小游戏
### 射箭
 - 游戏介绍：对一个靶子射箭
 - 标题：射箭，介绍：瞄准靶子按空格键发射箭矢吧！
 - 游玩描述：靶子在正中间，下面是弓箭，瞄准做圆弧形左右晃动，按空格键射箭，箭矢击中靶子得1分，一共5发箭矢
 - 游戏参数：
 - - 晃动速度，数值越高，晃动越低，最低是1秒一个来回，弧度是60度。计算公式默认是：
        晃动速度=1秒*（1-2*（我方速度-敌方速度）/（我方速度+敌方速度））
 - - 剩余时间，默认游戏时间是5秒，与敌我武力有关
        剩余时间=5秒*（1+（我方武力-敌方武力）/敌方武力）
 - 5发箭矢都射出去射偏或者射中，或者剩余时间到，游戏结束无法再发射箭矢
 - 结算：击中靶子的次数为得分，返回得分
